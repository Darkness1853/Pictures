# Правила участия в проекте

## Как сообщить об ошибке
- Проверьте, нет ли уже такого Issue
- Опишите шаги для воспроизведения

## Как предложить новую функцию
- Сначала создайте Issue для обсуждения
- Только потом пишите код

## Настройка окружения
- Какие версии Python/Node.js нужны
- Как установить зависимости

## Правила кода
- Используйте PEP8 (для Python) / Prettier (для JS)
- Называйте ветки: feature/название или fix/название

## Commit messages
- Формат: "feat: добавил кнопку" или "fix: исправил баг"

## Pull Request
- Прогоните тесты локально
- Дождитесь проверки от двух человек
